My Expenses решает конкретную задачу — быстрый, приватный и понятный учёт личных расходов без навязанных подписок и облачных сервисов. Данные не покидают устройство. Импорт банковской выписки — секунды. Автокатегоризация — на устройстве. Без регистрации, без backend, без компромиссов.

Простое объяснение

Приложение запоминает ваши исправления.

Исправили категорию — запомнил. В следующий раз тот же мерчант — правильная категория. Всё происходит на вашем телефоне. Никакой облачной нейросети. Локальный текстовый матчинг, который улучшается с использованием.

✎ Вы исправляете "LIDL" → Продукты
🧠 Приложение сохраняет правило локально
✔ "LIDL PARIS" → Продукты
Технические детали
P1Правила мерчантов

Точное / фразовое совпадение по нормализованному описанию. Высший приоритет.

P2Словарь правил

Словарь ключевых слов по категориям с весовыми приоритетами.

P3Локальный ML-классификатор

Bag-of-words + n-grams, косинусное сходство. На истории пользователя.

—Фоллбэк

Низкая уверенность → остаётся без категории для ручной проверки.

Локальная обработка

Все данные обрабатываются и хранятся на устройстве. Ничего не покидает ваш телефон.

Нет backend API. Нет облака. Нет серверной обработки.

Без регистрации

Никакого email, телефона, аккаунта. Установите и пользуйтесь.

CSV-файлы читаются на устройстве, никогда не загружаются на сервер.
